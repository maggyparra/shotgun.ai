# Shotgun.ai

Navigation app with Google Maps, Convex, and voice copilot.

**The app lives in [John's Folder](./John's%20Folder/).** Run and develop from there:

```bash
cd "John's Folder"
npm install
npx convex dev   # one terminal
npm run dev     # another terminal
```

See **[John's Folder/README.md](./John's%20Folder/README.md)** for full setup and **[John's Folder/CONVEX.md](./John's%20Folder/CONVEX.md)** for Convex.
